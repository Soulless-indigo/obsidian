### EX1

```scala 
object Main {
  def main(args: Array[String]): Unit = {
    // Créer une Range de 10 à 100
    val range = 10 to 100

    // Filtrer pour obtenir uniquement les nombres impairs
    val oddNumbers = range.filter(_ % 2 != 0)

    // Imprimer les nombres impairs
    oddNumbers.foreach(println)
  }
}
```
